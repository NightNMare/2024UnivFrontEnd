학습 내용 요약

챕터(제목) 소제목 내용 메모
HTML 기초 기초적인 태그들 HTML을 구성하는 기본적인 태그들 문서 구조를 만드는 핵심 요소
하이퍼링크 <a> : 하이퍼링크 생성 href 속성에 URL 삽입
이미지 <img> : 이미지 삽입 src 속성에 이미지 경로 삽입
테이블 <table> : 테이블 생성 표 형식의 데이터 구조화 가능
오디오 및 비디오 <audio>, <video> : 멀티미디어 추가 controls 속성으로 플레이어 제공
iframe <iframe> : 다른 웹 페이지 삽입 외부 페이지 삽입
입력 양식 <form> : 사용자가 서버에 데이터를 전송 방식 GET은 쿼리스트링 방식, POST는 암호화 방식으로 서버에 데이터 전달
CSS 기초 선택자 CSS 선택자 이해 및 응용. id, class, 태그 선택자 등 사용
박스 모델 박스 모델, 요소의 크기와 배치 마진, 패딩, 테두리 등
레이아웃 페이지 구조 설계 inline, block, inline-flex 등
JavaScript 기초 기본 특징 기본적인 특징과 웹에서의 역할 동적 웹 페이지 구현
변수 및 자료형 자료형 무관 var, let, const 사용 var는 재할당, 재선언 가능
let은 재할당 가능, 재선언 불가능
const 재할당, 재선언 불가능
